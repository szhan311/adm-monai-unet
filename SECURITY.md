# Security Policy

## Supported Versions

Security fixes are handled on the latest released version.

## Reporting A Vulnerability

Please report vulnerabilities privately by opening a GitHub security advisory:

https://github.com/szhan311/adm-monai-unet/security/advisories/new

If security advisories are not enabled for the repository, contact the
maintainer directly before publishing details.

## Scope

This package provides model code and experiment scripts. Do not treat model
outputs as clinical advice or diagnostic decisions without independent
validation and domain review.
